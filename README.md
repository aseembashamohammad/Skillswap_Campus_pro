# SkillSwap Campus Pro

## 📌 Project Description
SkillSwap Campus Pro is a Java-based console application that helps students connect with mentors based on their skills.

## 🚀 Features
- Add Student
- Display Students
- Search Mentor by Skill
- Update Student
- Delete Student
- File Handling (Data Storage)

## 🛠 Technologies Used
- Java
- File Handling (FileWriter, BufferedReader)
- ArrayList

## 🎯 Purpose
This project is built for learning Java OOP concepts and file handling for placement preparation.
